Download

Windows: After Downloading

1.Double click that .exe file to install it

Linux:After Downloading

1.Uncompress it:

sudo tar xvf langenius-linux-amd64.tar.bz2 -C /opt

2.Install it

cd /opt/langenius

sudo ./install.sh

3.Make it executable anywhere

sudo ln -s /opt/langenius/langenius /usr/bin

4.Now you can share your file on LAN !

langenius ./a.txt
