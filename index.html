下载

Windows: 安装步骤

1.双击下载的文件即可安装

Linux: 安装步骤

1.把下载的文件解压:

sudo tar xvf langenius-linux-amd64.tar.bz2 -C /opt

2.安装

cd /opt/langenius

sudo ./install.sh

3.让它可以在任何位置被执行

sudo ln -s /opt/langenius/langenius /usr/bin

4.开始分享你的文件到局域网上面吧!

langenius ./a.txt
